## 🚀 Featured Projects

### 🛸 [Autonomous Delivery Drone (UAV) System](#)
* **Tech:** C/C++, Pixhawk, ESP32-CAM, Raspberry Pi 5, GPS, Depth Sensors
* **Role:** Hardware & Sensor Integrator
* **Description:** Engineered a robust gripper arm (2 kg payload capacity) and integrated complex hardware modules (GPS M10, 3D Camera) to enable real-time obstacle detection and stable power distribution. *(Top 15 Finalist - UAV Innovation Cup PV GAS 2025)*.

### ⌚ [Federated Learning Wearable IoT System](#)
* **Tech:** Python, C++, Edge Devices, Linux
* **Role:** Edge Inference & Alert Engineer
* **Description:** Implemented a real-time inference loop on edge devices to process Federated Learning models. Engineered dynamic thresholding and alert logic based on ROC data analysis to trigger immediate local hardware notifications.

### 🅿️ [Smart Parking IoT & Backend System](#)
* **Tech:** Node.js, Express, TypeScript, MQTT, PostgreSQL, Prisma
* **Role:** Backend Developer
* **Description:** Architected a robust RESTful API and a real-time data pipeline using the MQTT protocol to process continuous data streams from local IoT sensor gateways. Managed multi-role users (RBAC) and complex dynamic pricing policies.

### 🏢 [Timesheet Management Platform (Monorepo)](#)
* **Tech:** TypeScript, Node.js, Express, React, PostgreSQL, Prisma
* **Role:** Full-stack Developer
* **Description:** Architected a production-ready Monorepo using Turbo and pnpm for an auditing firm. Implemented secure JWT authentication and configured an automated CI/CD deployment pipeline to Render.com.
